# 2ASIR-TFG
Servidor Flask dockerizado junto con un Cliente (Tetris) escrito en Python. Tiene guardado de puntuación, ranking global y página web para visualizar los rankings.

# Explicación de la elección del TFG
Aunque mi ciclo formativo no se centra en la Programación Orientada a Objetos, sí que se centra en la creación de infraestructuras y en todo lo relacionado con redes. <br/>
Dicho esto, creo que mi TFG es un claro ejemplo de una situación en la vida real donde necesitamos desplegar servidores y visualizar su contenido en varias plataformas, ya sea mediante páginas web, aplicaciones, etc...<br/>
Es por eso que he elegido este proyecto como mi TFG ya que demuestra mi conocimiento sobre desplegar servidores y dockerizarlos, creación de código en Python y la conectividad entre ambos (API).
